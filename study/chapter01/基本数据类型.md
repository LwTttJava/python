# 变量
标准数据类型

Python 定义了一些标准类型，用于存储各种类型的数据。

Python有五个标准的数据类型：


Numbers（数字）
        
        int（有符号整型）
           
        long（长整型，也可以代表八进制和十六进制）

        float（浮点型）

        complex（复数）
String（字符串）

List（列表）

Tuple（元组）

Dictionary（字典）

## Numbers数字类型

### int
```python
    a = 10
```

### String
```python
str = 'Hello World!'
print str           # 输出完整字符串
print str[0]        # 输出字符串中的第一个字符
print str[2:5]      # 输出字符串中第三个至第六个之间的字符串
print str[2:]       # 输出从第三个字符开始的字符串
print str * 2       # 输出字符串两次
print str + "TEST"  # 输出连接的字符串
```
