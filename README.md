python 创建对象之后会调用__init__方法
所以此方法又叫构造函数，常用来初始化对象

```python
__init__(第一个参数一定传调用实例,python中常用self表示,其他是属性看自己命名添加)
示例
__init__(self, data, length=None)
说明：length=None，这种写法(默认写法)是不穿length值 默认为None
```
